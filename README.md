# Wazuh Detection Pack

Custom detection rules for Wazuh based on Sysmon and Windows Security Events.

This repository is part of my Home SOC Lab and focuses on Detection Engineering.

## Environment

- Wazuh SIEM
- Windows Server (Active Directory)
- Windows 10 Workstation
- Sysmon
- Kali Linux (Attack Simulation)

## Structure

rules/
tests/
screenshots/
docs/

## Current detections

- net user
- whoami
- hostname
- ipconfig
- systeminfo

More detections will be added over time.
